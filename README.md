# bsl-sign-recognition
This is a Python project for recognising isolated British Sign Language signs using a webcam, and converting them into text.

## Current status

Project setup and research.

## Planned stages

- Capture webcam footage
- Detect hand landmarks
- Collect labelled sign data
- Train a classification model
- Display real-time predictions
- Evaluate the system on different users

## Scope

This project will use BSL (British Sign Language). It will initially only recognise a small amount of vocabulary. The goal of this project so far is to convert webcam input into text. The first prototype will use hand landmarks. The system will be evaluated on people whose recordings were not used for training. This project is not intended to replace interpreters or provide complete BSL translation.
